時短パスタの代表メニュー（カテゴリ別）

1. アーリオオーリオ

にんにくとオリーブオイルで作る王道の時短パスタ。

  • スパゲッティ 160〜180g（1人分〜1.5人分）
  • にんにく 2〜3片（薄切りまたはみじん切り）
  • オリーブオイル 大さじ3〜4
  • 赤唐辛子 1本（種を取るか輪切りでお好みで）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に適量
  • 黒胡椒 少々
  • パセリ（みじん切り） 適量（あれば）
  • 茹で汁 おたま1杯分（乳化に使う）
下準備

大きめの鍋にたっぷりの水（パスタ100gあたり約1Lが目安）を入れて沸かす。沸騰したら塩を加える（海水より少し薄い塩味が目安）。にんにくは好みの切り方にしておく。赤唐辛子は種を抜くと辛さが控えめになる。

作り方（手順）
  1. パスタを茹でる： 沸騰した湯にスパゲッティを入れ、袋表示の茹で時間より1分短めに茹で始める（仕上げで火を入れるため）。時々混ぜて麺がくっつかないようにする。
  2. オイルを温める： フライパンにオリーブオイルを入れて中弱火にかける。にんにくと赤唐辛子を入れ、焦がさないようにゆっくり香りを出す（にんにくが薄く色づき始めたらOK）。
  3. 茹で汁を用意： パスタが茹で上がる直前におたま1杯分の茹で汁を取っておく。
  4. パスタと合わせる： 茹で時間が終わったら湯切りをし、フライパンのオイルに直接加える。火は中火〜中強火に上げる。茹で汁を少しずつ加えながら手早く混ぜ、オイルと乳化させてソースを絡める。
  5. 味を整える： 塩と黒胡椒で味を調整し、最後にみじん切りのパセリを散らす。好みでレモン汁少々や粉チーズをかけてもOK。
  6. 盛り付け： 皿に盛り、にんにくチップを上に飾ると食感が出て見た目も良くなる。
コツとバリエーション
  • にんにくの火加減： 強火だとすぐ焦げるので中弱火でじっくり香りを出す。
  • 乳化を意識： 茹で汁のデンプンでオイルを乳化させるとソースが麺にしっかり絡む。
  • 具を足す場合： ベーコン、アンチョビ、きのこ、ほうれん草などを加えると満足感が上がる。具を入れる場合は先に炒めておく。
  • 辛さの調整： 赤唐辛子の量で調整。唐辛子をオイルでじっくり温めると辛みが穏やかになる。

2. しめじとバジルの冷製パスタ

茹でて冷やすだけの簡単さっぱりレシピ。

  • スパゲッティ 160〜180g
  • しめじ 1パック（石づきを落として小房に分ける）
  • プチトマト 6〜8個（半分またはくし切り）
  • バジルの葉 10枚前後（手でちぎる）
  • オリーブオイル 大さじ3
  • レモン汁 小さじ1〜2（好みで調整）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に適量
  • 黒胡椒 少々
  • にんにく（省略可） 1片（みじん切りまたはすりおろし）
  • 茹で汁 おたま1杯分（必要なら）
下準備

大きめの鍋にたっぷりの水を入れて沸かす。沸騰したら塩を加える。しめじは小房に分け、プチトマトは洗って切る。バジルは食べる直前まで冷蔵して香りを保つ。

作り方（手順）
  1. パスタを茹でる： 沸騰した湯でスパゲッティを袋表示どおりに茹でる。アルデンテが好みの場合は表示時間マイナス30秒〜1分を目安にする。
  2. しめじを下茹でする： 別の小鍋でさっと熱湯を通すか、フライパンで軽くオイルを引いて塩少々で炒め、余分な水分と生臭さを飛ばす。冷製にするので加熱は短めでシャキ感を残すとよい。
  3. 冷水で締める： 茹で上がったパスタは冷水でしっかり冷やし、ザルに上げて水気を切る。氷水で冷やすとさらに食感が締まる。
  4. ドレッシングを作る： ボウルにオリーブオイル、レモン汁、にんにく（使う場合）、塩、黒胡椒を入れてよく混ぜる。味を見て調整する。
  5. 混ぜ合わせる： 冷えたパスタをドレッシングのボウルに入れ、しめじ、プチトマト、ちぎったバジルを加えて優しく和える。必要ならおたま1杯分の茹で汁で全体をなじませる。
  6. 冷やして味をなじませる： 盛り付ける前に冷蔵庫で10〜15分ほど冷やすと味が落ち着く。
  7. 盛り付け： 皿に盛り、仕上げにオリーブオイルを軽く回しかけ、黒胡椒を挽き、バジルの葉を飾る。好みでパルメザンチーズ少々を削って添える。
コツとバリエーション
  • 冷たさと食感： 茹で上がりを素早く氷水で締めるとコシが保てる。
  • 水っぽさ対策： パスタはしっかり水気を切り、ドレッシングは少なめから調整する。
  • 旨味の追加： アンチョビペースト少量、または刻んだオリーブを加えると味に深みが出る。
  • タンパク質を足す： 蒸し鶏、ツナ、モッツァレラチーズを加えると主菜になる。
  • 作り置き： 食べる直前にバジルを混ぜると香りが飛ばずに美味しい。

3. 明太子バター

明太子とバターで濃厚に仕上げる短時間メニュー。

  • スパゲッティ 160〜180g
  • 明太子 1腹（約50〜70g）
  • 無塩バター 20〜30g
  • 生クリームまたは牛乳 大さじ1〜2（好みで）
  • 醤油 小さじ1（風味付け）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に少々
  • 黒胡椒 少々
  • 刻み海苔または刻みねぎ 適量（仕上げ用）
  • レモンまたは酢 少々（好みで酸味を加える）
  • 茹で汁 おたま1杯分（必要なら）
下準備

明太子は腹を開いて中身をスプーンで取り出す。辛さが強い場合は薄皮の一部を取り除くとマイルドになる。バターは常温に戻しておくと混ざりやすい。

作り方
  1. パスタを茹でる： 沸騰した塩入りの湯でスパゲッティを袋表示通りに茹でる。アルデンテが好みなら表示時間より少し短めに。
  2. ソースを準備する： ボウルに明太子の中身とバターを入れ、滑らかになるまでよく混ぜる。生クリームを加える場合はここで混ぜ、醤油を少量加えて風味を整える。
  3. 茹で汁で調整： パスタが茹で上がる直前におたま1杯分の茹で汁を取り分ける。茹で上がったパスタを湯切りし、熱いうちにボウルの明太子バターと合わせる。茹で汁を少しずつ加え、乳化させながら滑らかにする。
  4. 味を整える： 必要なら塩で調整し、黒胡椒を軽く挽く。味が濃い場合は生クリームや茹で汁で伸ばす。好みでレモンを数滴落として酸味を足すと後味がさっぱりする。
  5. 盛り付け： 皿に盛り、刻み海苔または刻みねぎをたっぷり散らす。風味を増したければ追加でバターを少量のせる。
コツとバリエーション
  • 明太子の処理： 薄皮ごと混ぜるとプチプチ感が残る。滑らかにしたければ薄皮は除く。
  • 塩気の調整： 明太子自体に塩分があるため、茹で汁やクリームで調整するのが安全。
  • 濃厚さの調整： バターを増やすとコクが出る。軽めにしたければ牛乳で代用する。
  • 合わせ技： しめじやほうれん草を軽く炒めて加えると食感と栄養バランスが良くなる。
  • 時短のコツ： ボウルでソースを先に作っておき、パスタが茹で上がる直前に合わせるだけにしておくとスムーズ。

4. ツナの和えるパスタ

缶詰を活用した即席ソースで時短。冷蔵庫にある材料で手早く作れて、ランチや夜食にぴったりです。

  • スパゲッティ 160〜180g
  • ツナ缶（オイル漬けまたは水煮） 1缶（70〜100g）
  • 玉ねぎ 1/4個（薄切り）
  • マヨネーズ 大さじ1〜2
  • 醤油 小さじ1
  • レモン汁 小さじ1（または酢）
  • オリーブオイル 大さじ1
  • 黒胡椒 少々
  • 刻みねぎまたはパセリ 適量（仕上げ用）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に少々
  • お好みで具材（コーン、刻みピクルス、ゆで卵、刻み海苔など）
下準備

玉ねぎは薄切りにして辛味を和らげたい場合は水にさらすか軽く塩もみする。ツナは缶汁を軽く切る（オイル漬けはオイルを少し残すとコクが増す）。大きめの鍋でたっぷりの湯を沸かし、塩を入れておく。

作り方（手順）
  1. パスタを茹でる： 沸騰した湯でスパゲッティを袋表示どおりに茹でる。茹で上がる直前におたま1杯分の茹で汁を取る。
  2. ソースを作る： 大きめのボウルにツナ、マヨネーズ、醤油、レモン汁、オリーブオイルを入れてよく混ぜる。玉ねぎを加えて軽く和える。味を見てマヨネーズや醤油で調整する。
  3. パスタと和える： 茹で上がったパスタをしっかり湯切りし、熱いうちにボウルへ入れて素早く混ぜる。とろみが欲しい場合は茹で汁を少量ずつ加えて調整する。
  4. 仕上げ： 黒胡椒を挽き、刻みねぎやパセリを散らす。お好みでコーンや刻みピクルス、ゆで卵をトッピングする。
  5. 冷製にする場合： 混ぜたあと粗熱を取ってから冷蔵庫で10〜20分冷やすと味が落ち着き、夏場はよりさっぱり食べられる。
コツとバリエーション
  • ツナの種類： オイル漬けはコクが出る。水煮はあっさり仕上がるのでレモンやオリーブオイルで風味を補う。
  • 食感を足す： 玉ねぎの代わりにセロリや刻んだきゅうりを使うとシャキシャキ感が出る。
  • 和風アレンジ： マヨネーズを減らしてみりん少々とだし醤油を加えると和の味わいになる。
  • ボリュームアップ： 蒸し鶏や枝豆、ゆでたブロッコリーを加えると栄養バランスが良くなる。
  • 時短のコツ： ソースは先に作っておき、パスタが茹で上がったらすぐ和えるだけにすると調理が一気に早くなる。

5. ひき肉ときのこパスタ

ひき肉ときのこで作る時短パスタ。旨味たっぷりで食べごたえがあり、忙しい日の主菜にぴったりです。

  • スパゲッティ 160〜180g
  • 合いびき肉 100g
  • きのこ（しめじ、しいたけ、まいたけなど）合計100〜150g（石づきを取りほぐす）
  • 玉ねぎ 1/4個（みじん切り）
  • にんにく 1片（みじん切り）
  • オリーブオイルまたはサラダ油 大さじ1〜2
  • 白ワイン（または酒） 大さじ1
  • トマト缶（カット）またはトマトソース 100〜150g（トマト味にする場合）
  • 醤油またはウスターソース 小さじ1（コク出し）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に適量
  • 黒胡椒 少々
  • パセリまたはバジル 適量（仕上げ用）
  • 茹で汁 おたま1杯分（必要に応じて）
下準備

大きめの鍋にたっぷりの水を入れて沸かす。沸騰したら塩を加える。きのこは食べやすくほぐし、玉ねぎとにんにくは刻んでおく。トマトを使う場合は缶を開けて用意する。

作り方（手順）
  1. パスタを茹でる： 沸騰した湯でスパゲッティを袋表示どおりに茹でる。茹で上がる直前におたま1杯分の茹で汁を取っておく。
  2. ひき肉ときのこを炒める： フライパンにオリーブオイルを熱し、にんにくを弱火で香りが出るまで炒める。玉ねぎを加えて透き通るまで中火で炒める。
  3. ひき肉を加える： 合いびき肉を加えてほぐしながら炒め、色が変わったら余分な油を軽く拭き取る。白ワインを振ってアルコールを飛ばす。
  4. きのこを加える： きのこを加えて強めの中火で水分を飛ばしつつ炒める。きのこから旨味が出てきたら、トマトを使う場合はここでトマト缶を加える。
  5. 味付け： 醤油またはウスターソースでコクを足し、塩と黒胡椒で味を整える。トマトソースが濃い場合は茹で汁で濃度を調整する。
  6. パスタと合わせる： 茹で上がったパスタを湯切りしてフライパンに加え、中火で1分ほど手早く和える。茹で汁を少しずつ加えてソースを乳化させ、麺にしっかり絡ませる。
  7. 盛り付け： 皿に盛り、刻んだパセリやバジルを散らす。好みで粉チーズを振っても美味しい。
コツとバリエーション
  • きのこの水分管理： きのこは炒めすぎると水っぽくなるので、最初は強めの火で素早く炒めて余分な水分を飛ばすと味が濃くなる。
  • ひき肉の旨味を引き出す： 焦げ目が少し付くくらいしっかり焼くと香ばしくなる。
  • 和風アレンジ： トマトを使わずに醤油・みりん・和風だしで味付けするとご飯にも合う和風パスタに変身する。
  • ボリュームUP： 茹でたほうれん草、ブロッコリー、または目玉焼きを乗せると満足感が増す。
  • 時短の工夫： ひき肉は冷凍しておけば時短で解凍しながら炒められる。ソースは先に作って温め直すだけでも楽。

6. 納豆パスタ

和風味の朝食に最適、混ぜるだけで栄養満点の時短メニュー。納豆のねばりと調味料がパスタによく絡みます。

  • スパゲッティ 160〜180g
  • 納豆 1パック（タレ付きの場合はタレも使用）
  • しょうゆ 小さじ1〜2
  • ごま油 小さじ1（風味付け）
  • 刻みねぎ 大さじ1〜2
  • 刻み海苔 適量（仕上げ用）
  • 練りからしまたは一味唐辛子 少々（好みで）
  • 卵黄 1個（お好みで）
  • 白ごま 少々（仕上げ用）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に少々
  • 茹で汁 おたま1杯分（必要なら）
下準備

納豆はパックの中で付属のタレとよく混ぜておく。好みで刻みねぎや練りからしを最初に混ぜておくと味がなじみやすい。大きめの鍋にたっぷりの水を入れて沸かし、塩を加える。

作り方（手順）
  1. パスタを茹でる： 沸騰した塩入りの湯でスパゲッティを袋表示どおりに茹でる。アルデンテが好みなら表示時間より少し短めに茹でる。
  2. 納豆ソースを用意： ボウルに納豆を入れ、付属のタレを混ぜる。そこにごま油としょうゆを加え、粘りが出るまでよく混ぜ合わせる。味見をしてしょうゆで微調整する。お好みで練りからしを加える。
  3. 茹で汁を取る： パスタが茹で上がる直前におたま1杯分の茹で汁を取り分ける。
  4. 和える： 茹で上がったパスタを湯切りして熱いうちに納豆のボウルに入れる。茹で汁を少量ずつ加えながら手早く混ぜ、納豆をパスタに均一に絡める。茹で汁は粘度と味の調整用に加減する。
  5. 仕上げ： 器に盛り、刻みねぎ、刻み海苔、白ごまを散らす。好みで中央に卵黄を落とすとまろやかさが増す。ピリッとさせたい場合は一味唐辛子を振る。
コツとバリエーション
  • 温かさのバランス： 熱々のパスタに納豆を合わせると香りが立ち、よりおいしくなる。冷製にする場合はパスタを氷水で締めてから和える。
  • 粘りを活かす： 納豆は調味料を混ぜてからしっかり粘らせておくとよく絡む。
  • 旨味の追加： ほんの少量の顆粒だしや白だしを加えると深みが出る。
  • 食感のアクセント： 大葉や刻んだみょうが、刻んだ梅干し、刻んだ漬物を足すと爽やかな変化が出る。
  • タンパク質強化： ツナやほぐした鶏ささみを混ぜても相性が良い。
  • 時短の工夫： 納豆ソースは先に混ぜておき、パスタが茹で上がったらすぐ和えるだけにしておくと手早く仕上がる。

7. レモンペッパーのさっぱりパスタ

レモンで爽やかに仕上げる短時間メニュー。朝食や軽めのランチにぴったりです。

  • スパゲッティ 160〜180g
  • レモン 1/2個（皮のすりおろしと果汁）
  • オリーブオイル 大さじ2〜3
  • にんにく 1片（薄切りまたはみじん切り、好みで省略可）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に適量
  • 粗挽き黒胡椒（レモンペッパー） 小さじ1〜2（好みで調整）
  • パルメザンチーズ 大さじ1（仕上げ、好みで）
  • バジルまたはイタリアンパセリ 適量（飾り用）
  • 茹で汁 おたま1杯分（乳化用）
  • お好みのトッピング（ツナ、蒸し鶏、焼いたズッキーニ、プチトマトなど）
下準備

大きめの鍋にたっぷりの水を入れて沸かす。沸騰したら塩を入れる。レモンはよく洗い、皮を細かくすりおろしてから果汁を絞る。にんにくは好みの切り方にする。

作り方（手順）
  1. パスタを茹でる： 沸騰した塩入りの湯でスパゲッティを袋表示どおりに茹でる。茹で上がる直前におたま1杯分の茹で汁を取っておく。
  2. オイルで香りを出す： フライパンにオリーブオイルを入れて中弱火で温め、にんにくを入れて焦がさないようにゆっくり香りを出す。にんにくの香りが立ったら火を弱める。
  3. レモンと合える準備： ボウルにオリーブオイル少量（分量の一部）とレモン果汁、レモンの皮すりおろし、粗挽き黒胡椒、塩を入れて混ぜておく。
  4. パスタと合わせる： 茹で上がったパスタを湯切りし、フライパンのオイルか直接先ほどのボウルに入れる。おたま1杯分の茹で汁を少しずつ加えながら手早く混ぜ、ソースを乳化させて麺に絡める。
  5. 味を調整： レモン汁や黒胡椒で酸味・辛味を調整し、必要なら塩を足す。パルメザンチーズを加える場合はこのタイミングでさっと混ぜるとコクが出る。
  6. 盛り付け： 皿に盛り、仕上げにレモンの皮を少量振り、刻んだバジルやイタリアンパセリを散らす。好みでトッピングを追加する。
コツとバリエーション
  • レモンの使い分け： 皮は香り付けに少量だけ使い、果汁で酸味を調整する。皮を多く使うと苦味が出るので注意する。
  • 辛味の調整： 粗挽き黒胡椒を増やすと「レモンペッパー」感が強くなる。辛さが苦手なら少なめに。
  • さっぱり軽めに： にんにくを省略し、オリーブオイルを控えめにするとより朝向けのさっぱり味になる。
  • ボリュームアップ： ツナや蒸し鶏、アボカドを加えると満足感が増す。トマトやルッコラを添えると彩りと食感が良くなる。
  • 作り置きのポイント： レモンの風味は時間とともに飛びやすいので、作り置きする場合は食べる直前にレモンを絞って合わせると香りが活きる。

8. ペペロンチーノ（時短）

にんにくと鷹の爪のシンプルスピードメニュー。短時間で香り高く仕上げるコツを押さえれば、家庭でもプロの味に近づきます。

  • スパゲッティ 160〜180g
  • にんにく 2片（薄切りまたはみじん切り）
  • 鷹の爪（赤唐辛子） 1本（輪切りまたは手で折る）
  • オリーブオイル 大さじ3（エキストラバージン推奨）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に少々
  • 粗挽き黒胡椒 お好みで少々
  • 茹で汁 おたま1杯分（乳化と調整用）
  • パセリまたは刻みバジル 適量（仕上げ用）
  • にんにくチップ（オプション） 少々（飾りと食感用）
下準備

大きめの鍋にたっぷりの水を入れて沸かす。沸騰したら塩を加える。にんにくは好みの切り方にしておき、鷹の爪は種を抜くと辛さがまろやかになる。フライパンはパスタを合わせる直前まで弱火にしておく。

作り方（手順）
  1. パスタを茹でる： 沸騰した湯にスパゲッティを入れ、袋表示の茹で時間どおりに茹でる。表示時間の30秒〜1分前にやや固めで上げると仕上げでちょうどよくなる。茹で上がる直前におたま1杯分の茹で汁を取る。
  2. にんにくオイルを作る： フライパンにオリーブオイルを入れ、中弱火で温める。にんにくと鷹の爪を入れ、焦がさないようにゆっくり香りを出す（にんにくが薄く色づくまで）。にんにくをカリッとさせたい場合は別で低温で揚げるか、オーブンで焼いてにんにくチップを作る。
  3. パスタを合わせる： 茹で上がったパスタを湯切りし、フライパンに直接入れる。強めの中火にしておたまの茹で汁を少しずつ加えながら手早く和え、オイルと乳化させて麺にソースを絡める。
  4. 味を調整： 塩で味を整え、好みで粗挽き黒胡椒を振る。オリーブオイルを最後に少量垂らすと風味が立つ。
  5. 盛り付け： 皿に盛り、刻んだパセリやバジルを散らす。仕上げににんにくチップを載せると食感と見た目が良くなる。
コツとバリエーション
  • にんにくの火加減： にんにくは中弱火でゆっくり加熱し、薄く色づいたら取り出す。強火だと苦味が出るので注意。
  • 乳化を意識： 茹で汁のデンプンでオイルを乳化させるとソースが麺にしっかり絡む。
  • 辛さ調整： 鷹の爪の量や種の有無で調整。辛さを抑えたい場合は種を除くか、鷹の爪を一旦オイルで温めて取り出す。
  • 時短ワザ： ソースはフライパンで先に作っておき、パスタが茹で上がったらすぐ和えるだけにすると調理時間を短縮できる。
  • 具の追加： ベーコンやアンチョビ、しめじなどを加えるとボリュームが出る。野菜を足す場合は先に炒めて水分を飛ばしてから合わせる。
  • 仕上げの香り： 仕上げにレモンの皮少々をすりおろすと爽やかさが加わる。

9. ツナマヨパスタ

ツナ＋マヨで子どもも喜ぶ簡単レシピ。手早く作れて味付けも安定するので、お弁当や夜食にも便利です。

  • スパゲッティ 160〜180g
  • ツナ缶 1缶（オイル漬けまたは水煮、70〜100g）
  • マヨネーズ 大さじ1.5〜2
  • しょうゆ 小さじ1（風味付け）
  • 粗挽き黒胡椒 少々
  • 刻みねぎまたはパセリ 大さじ1（仕上げ）
  • コーン 大さじ2（缶詰・冷凍どちらでも可、好みで）
  • 塩 パスタ茹で用に大さじ1程度、味調整用に少々
  • 茹で汁 おたま1杯分（必要なら）
  • レモン汁 少々（好みでさっぱりさを追加）
下準備

ツナは缶汁を軽く切る（オイル漬けはオイルを少し残すとコクが出る）。コーンは水気を切る。大きめの鍋にたっぷりの水を沸かし、塩を入れておく。

作り方（手順）
  1. パスタを茹でる： 沸騰した塩入りの湯でスパゲッティを袋表示どおりに茹でる。茹で上がる直前におたま1杯分の茹で汁を取っておく。
  2. ソースを作る： 大きめのボウルにツナ、マヨネーズ、醤油を入れてよく混ぜる。必要なら黒胡椒を少々加える。レモン汁を少量加えると味が締まる。
  3. パスタと和える： 茹で上がったパスタを湯切りして熱いうちにボウルへ入れ、コーンと刻みねぎを加えて素早く和える。とろみが欲しい場合は茹で汁を少量ずつ加えて調整する。
  4. 味を整える： 味を見て塩やマヨネーズで微調整する。子ども向けには黒胡椒を控えめにするか抜いてもよい。
  5. 盛り付け： 皿に盛り、仕上げに刻みねぎやパセリを散らす。お好みで刻み海苔やゆで卵スライスを添えると見た目と満足感がアップする。
コツとバリエーション
  • 油の量： オイル漬けツナの油を少し残すとコクが増すが、さっぱりが好みならしっかり切る。
  • 食感アップ： きゅうりの角切りやセロリのみじん切りを加えるとシャキシャキ感が出る。
  • 和風アレンジ： マヨネーズを減らしてめんつゆ少量と和えても美味しい。
  • お弁当向け： 冷めても美味しいが、水分が出やすいので少し固めに和えるとよい。
  • 時短ワザ： ソースを事前に作っておき、パスタが茹で上がったらすぐ和えると手早く完成する。
